# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-30

### Added
- Initial release of Mermaid to Draw.io converter
- Support for multiple diagram types: flowcharts, gantt charts, kanban boards, mindmaps, org charts, pie charts, sequence diagrams, SWOT analysis, timelines, user journey maps, and more
- Command-line interface with Playwright integration
- Comprehensive guidelines and templates for various diagram types
- Cheat sheet and contributing guidelines

### Dependencies
- Added cheerio for HTML parsing
- Added commander for CLI functionality
- Added dagre for graph layout
- Added date-fns for date handling
- Added fs-extra for enhanced file operations
- Added playwright for browser automation
- Added xmlbuilder2 for XML generation
