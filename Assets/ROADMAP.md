# Roadmap

This document outlines the planned features and improvements for the Mermaid to Draw.io converter.

## Upcoming Features

### Version 1.1.0
- [ ] Add support for state diagrams
- [ ] Add support for class diagrams
- [ ] Add support for entity relationship diagrams
- [ ] Improve error handling and validation
- [ ] Add progress indicators for large conversions

### Version 1.2.0
- [ ] Add batch conversion feature
- [ ] Add web interface for online conversion
- [ ] Add support for custom themes and styling
- [ ] Add export to multiple formats (PNG, SVG, PDF)
- [ ] Add integration with popular diagramming tools

### Version 2.0.0
- [ ] Complete rewrite for better performance
- [ ] Add support for all Mermaid diagram types
- [ ] Add real-time collaborative editing
- [ ] Add plugin system for custom converters
- [ ] Add API for programmatic usage

## Long-term Vision

### Version 3.0.0
- [ ] AI-powered diagram optimization
- [ ] Automatic diagram layout improvements
- [ ] Integration with major design tools (Figma, Sketch)
- [ ] Cloud-based conversion service
- [ ] Mobile app support
